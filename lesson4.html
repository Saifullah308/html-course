Working with Lists and Tables

Unordered Lists

<ul> creates a bulleted list. Each item is wrapped in <li> tags.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

Ordered Lists

<ol> creates a numbered list. You can change numbering with the type attribute.

<ol type="1"> <!-- Default: 1, 2, 3 -->
    <li>First item</li>
    <li>Second item</li>
</ol>

<ol type="A"> <!-- A, B, C -->
    <li>Item A</li>
    <li>Item B</li>
</ol>

<ol type="I"> <!-- Roman numerals -->
    <li>Step I</li>
    <li>Step II</li>
</ol>

Description Lists

<dl> creates a list of terms and descriptions.

<dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>
    <dt>CSS</dt>
    <dd>Cascading Style Sheets</dd>
</dl>

Nested Lists

You can nest lists inside other lists:

<ul>
    <li>Fruits
        <ul>
            <li>Apples</li>
            <li>Oranges</li>
        </ul>
    </li>
    <li>Vegetables
        <ul>
            <li>Carrots</li>
            <li>Broccoli</li>
        </ul>
    </li>
</ul>

Basic Tables

Tables are created with <table>, rows with <tr>, headers with <th>, and cells with <td>.

<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>Alice</td>
        <td>28</td>
    </tr>
    <tr>
        <td>Bob</td>
        <td>32</td>
    </tr>
</table>

Table Structure Elements

For better organization, tables can include:

<table>
    <caption>Monthly Sales</caption>
    <thead>
        <tr><th>Month</th><th>Sales</th></tr>
    </thead>
    <tbody>
        <tr><td>January</td><td>$1000</td></tr>
    </tbody>
    <tfoot>
        <tr><td>Total</td><td>$1000</td></tr>
    </tfoot>
</table>
